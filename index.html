🐧 Что это такое?

Дистрибутив Linux на базе Arch, старательно пытающийся адаптироваться под пользователя, который только пришел с Windows. Забудьте про страх терминала — мы автоматизировали рутину, оставив вам только скорость и эстетику.

Технологии под капотом

Почему LemonovOS работает быстрее и стабильнее других сборок.

⚡ Ядро Zen + ZRAM

Используем ядро linux-zen. Настроен динамический планировщик задач, а также ZRAM (с агрессивным swappiness=133). Забудьте про зависания браузера с 50 вкладками на 8 ГБ оперативки.

🍹 Установщик Juicer

Никакого Calamares. Мы написали собственный графический установщик Juicer на Python (PySide6). 12 простых шагов, умное автоопределение драйверов Nvidia/AMD и красивый лог установки.

🕰️ Btrfs Snapshots

Боитесь обновлять Arch? По умолчанию мы используем файловую систему Btrfs со встроенными авто-снапшотами. Сломали систему — просто выберите вчерашний снимок в меню GRUB.

🎮 Gaming OOTB

Готово к играм прямо из коробки. Оптимизированный сетевой стек (BBR), вшитые gamemode и поддержка кастомных версий Proton. Выжимаем максимум FPS без курения мануалов.

Как бесплатно разместить этот сайт на GitHub Pages.

Этот сайт состоит всего из одного файла index.html (и папки img/ для скриншотов). Идеальное место для его бесплатного хостинга с SSL сертификатом — GitHub Pages.

  1. Создай новый публичный репозиторий на GitHub. Например: LemonovOS-Web.
  2. Загрузи в этот репозиторий текущий файл index.html.
  3. Туда же загрузи папку img/ с твоими скриншотами (desktop-dark.jpg, terminal-dark.png и т.д.).
  4. Перейди в настройки репозитория: Settings -> Pages (в левом меню).
  5. В разделе "Build and deployment" выбери Source: Deploy from a branch.
  6. В выпадающем списке Branch выбери main (или master) и нажми Save.

Через пару минут твой сайт будет доступен по ссылке вида:
https://ТВОЙ_НИК.github.io/LemonovOS-Web/
